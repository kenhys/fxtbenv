## Requirements

* go 1.9.2 or later

## How to Install `fxtbenv`

```
$ git clone https://github.com/kenhys/fxtbenv $HOME/.fxtbenv
$ cd $HOME/.fxtbenv
$ make
$ source $HOME/.fxtbenv/scripts/fxtbenv
```
